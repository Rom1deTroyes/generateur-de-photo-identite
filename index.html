    • Générateur de photo d'identité

    Créez vos planches de photo d'identitée chez vous.

    Projet maintenu par karlito139 Hébergé sur les pages de GitHub — Theme par mattgraham

    Le logiciel

    Ce logiciel vous permetra de créer vos propres planche de photo d'identé soit en prenant une phot avec votre webcam, soit en important une photo de vous. Le resultat doit être conforme aux recommandations du ministère de l'interieur fraçais (verifiez quand même que tout concorde).

    Installation

    Pour installer ce logiciel récupérez l'installateur correspondant à votre system.

    Linux

    Vous pouvez récupérer le packet deb pous linux 32 bits ici.
    Vous pouvez récupérer le packet deb pous linux 64 bits ici.

    Windows

    Pour windows vous pouvez récupérer l'installateur ici.

    Sources

    Vous pouvez télécharger les sources de l'application directement sur github (bouton en haut à gauche de la page) ou en téléchargement directement l'archive en zip ou tar. Pour le compiler vous aurez besoin d'avoir installé opencv au préalable.
